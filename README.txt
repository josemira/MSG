# MSG

Weekday Counter
This Python program calculates the number of weekdays (Monday to Friday) between two given dates, inclusive of both the start and end dates.

Getting Started:

Prerequisites
1. Make sure you have Python installed on your system. This program was developed using Python 3.x.

Installing
1. Clone the repository or download the weekday_counter.py file.
2. Ensure you have Python installed. You can download it from python.org.

Usage
1. Open a terminal or command prompt.
2. Run the weekday_counter.py script with the start and end dates as arguments.
